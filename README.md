## What is NativeCode?

NativeCode is a native macOS coding IDE with a built-in AI agent that runs **entirely on your Mac**. No cloud. No API keys. No subscription. Your code never leaves your device.

- 🍎 **Native SwiftUI** — not Electron, not a web wrapper
- ⚡ **Powered by MLX** — Apple Silicon optimized inference
- 🧠 **Qwen3 models** — 8B and 14B in 4/6/8-bit quantizations
- 🔒 **100% local** — runs on `localhost:8000`, no telemetry
- 🆓 **Free forever** — no subscription, no account required
- 📝 **Monaco editor** — the same editor that powers VS Code
- 🛠️ **Agentic AI** — reads, edits, searches files with your approval

## Quick start

1. Download `NativeCode.dmg` from [nativecode.jakubhecht.sk](https://nativecode.jakubhecht.sk)
2. Drag NativeCode to your Applications folder
3. Open the app and follow the in-app setup (installs Rapid-MLX, downloads a model)
4. Start coding with your local AI agent

Full installation guide: [nativecode.jakubhecht.sk/#install](https://nativecode.jakubhecht.sk/#install)

## System requirements

- macOS 14.0 or later
- Apple Silicon Mac (M1 / M2 / M3 / M4) recommended
- 16 GB RAM minimum (32 GB recommended for larger models)
- ~5–15 GB free disk space (depending on chosen model)
